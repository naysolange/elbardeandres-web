El proceso

El video muestra distintas pruebas realizadas del prototipo de circuito electrónico

Los resultados de este proyecto se materializaron en la confección de la página web con las distintas secciones y el prototipo del arnés háptico. Si bien las plataformas correspondientes a las salas temáticas aún están en proceso de creación, apuntamos a tener una buena base con respecto a la página web, planteando a este proyecto como un punto de partida. Sin lugar a dudas, realizar un proyecto grupal que incluya un prototipo funcional en épocas de cuarentena ha sido todo un desafío. Elaborar una división de trabajo equitativa, coordinar horarios, enviar partes, recibir partes, realizar videollamadas todas las semanas fue la forma en la que pudimos lograr todo esto. Para cerrar creemos fielmente que este proyecto puede subsistir como una forma de comunicación innovadora en época pos pandemia, y continuaremos trabajado en ello.
